Tecnología de Información

Tecnologías con las que he trabajado

GIS Servidor (Nube / Dedicado)

Implementación de infraestructura GIS para publicación de servicios, visores web y gestión de datos geoespaciales.

Diseño e implementación de servidores GIS en entornos cloud o dedicados, orientados a rendimiento, disponibilidad y escalabilidad. La solución puede desplegarse sobre Linux (Ubuntu) o Windows Server, integrando componentes para publicación de mapas y consumo de datos por aplicaciones Web GIS.

  • Cliente: Presentación / Demo
  • Implementación: Infraestructura GIS
  • Stack: Docker · Tomcat · Oracle XE · Linux/Windows Server
  • Acceso: Controlado (demo en reunión)

Encuesta Georreferenciada Offline

Captura de datos en campo con QField (QGIS), sincronización y control de calidad de la información.

Implementación de flujos de levantamiento en campo para inventarios, replanteo y encuestas georreferenciadas. Permite trabajar offline, registrar atributos y geometrías, y luego sincronizar los datos a QGIS para validación y publicación.

  • Cliente: Capacitación / Implementación
  • Categoría: GIS Móvil / Levantamiento en campo
  • Herramientas: QGIS · QField · Formularios

Tendido de Fibra Óptica

Visualización y control del recorrido de red (canalizado, cámaras, postes) con panel y filtros operativos.

Desarrollo de un visor GIS para el seguimiento del tendido de fibra óptica, mostrando el recorrido de la red por canalizado, cámaras y postes. Incluye un panel de control con filtros por estado (proyectado / en ejecución / terminado) para apoyar la gestión y supervisión de avance.

  • Cliente: Presentación
  • Categoría: Telecomunicaciones
  • Enfoque: Control de avance · Inventario georreferenciado

Postes para Fibra Óptica

Planificación y soporte GIS para instalación de postes y despliegue de red.

Soporte GIS para la instalación de postes destinados al tendido de fibra óptica. Incluye análisis espacial, coordinación de permisos y documentación, asegurando trazabilidad y consistencia de la información del proyecto.

  • Cliente: Presentación
  • Categoría: Telecomunicaciones
  • Enfoque: Gestión territorial · Permisos · Inventario

API Catastral

Servicios backend para interoperabilidad GIS y exportación de datos en formatos estándar.

Implementación de una API orientada a procesos catastrales para generar y entregar paquetes de datos geoespaciales listos para análisis y publicación en plataformas GIS. Soporta exportación incremental y control de ejecución mediante auditoría/bitácora.

La documentación se gestiona mediante OpenAPI/Swagger, e incluye endpoints de verificación (health checks) para asegurar la estabilidad operativa y la conectividad con bases de datos.

  • Formatos: GeoPackage · GeoJSON
  • Documentación: OpenAPI / Swagger
  • Base de datos: Oracle · PostgreSQL
  • Funciones: Exportación incremental · Auditoría · Health checks
  • Acceso: Demo técnica disponible en reunión (acceso controlado)

GeoServer (Demo)

Publicación de servicios OGC (WMS/WFS) para consumo en visores, apps y plataformas GIS.

Configuración y publicación de capas geoespaciales mediante GeoServer, habilitando servicios OGC para interoperabilidad (WMS/WFS). Incluye administración de fuentes de datos, estilos (SLD) y soporte para consumo desde MapStore, QGIS y aplicaciones web.

  • Servicios: WMS · WFS
  • Categoría: Servicios OGC / Publicación GIS
  • Acceso: Demo disponible (según disponibilidad)
